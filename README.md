
# Spotify Clone 🎵

A responsive Spotify-inspired web UI built using HTML, CSS, and Bootstrap.

## 🚀 Features
- Responsive design for different screen sizes
- Sidebar navigation and music player layout
- Clean and modern UI inspired by Spotify

## 🛠️ Tech Stack
- HTML
- CSS
- Bootstrap

## 📌 Project Purpose
This project was built to improve my frontend development skills and understand real-world UI structuring.
